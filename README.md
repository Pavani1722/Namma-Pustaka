# Namma-Pustaka

Namma-Pustaka is a smart library assistant app for rural schools. It converts a small physical book shelf into a searchable digital catalog and borrowing register.

## Features

- Browse books by category in a digital shelf layout.
- Search by book name or author.
- Add a new book with a camera-based cover entry.
- Borrow or return books using a QR/code style screen.
- Show overdue books automatically in red.
- Add star ratings and one-line reviews.
- View a monthly reading leaderboard.
- Store catalog and transaction data locally using SQLite.

## Open In Android Studio

1. Open Android Studio.
2. Choose **Open**.
3. Select this project folder.
4. Let Gradle sync.
5. Run the app on an emulator or Android phone.

The project uses Java and Android XML/resources only, with no external runtime dependencies beyond the Android Gradle plugin.
